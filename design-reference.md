# Design Reference from Rdash.io

## Color Scheme
- Primary accent: Coral/Red (#FF4B4B or similar) - used for CTAs and highlights
- Secondary: Dark navy/black for text
- Background: Clean white/light gray
- Professional construction industry feel

## Key Features Observed
1. Project Management - Survey to Designs, BOQs, Vendor allocation
2. Vendor Management - Suppliers, contractors, rate contracts
3. Task Management - Team tasks, reminders, approvals, issues
4. AI-powered assistance

## Value Propositions
- 100% Visibility - Real-time project data
- Avoid Delays - Streamlined communication
- Reduce Cost - Budget control through approvals
- Quality Control - Audits and checklists

## Design Style
- Clean, professional, modern
- Card-based layouts
- Clear hierarchy with bold headings
- Construction imagery (workers, sites)
- Testimonial sections for trust
- Simple navigation structure

## UI Elements
- Rounded buttons with solid colors
- Card components with subtle shadows
- Clear section dividers
- Professional typography
- Mobile-responsive design
